# 🎯 Performance Management System (PMS)
Production SaaS — React 18 + Vite + Zustand + React Query + Supabase + Recharts

## Quick Start
1. `npm install`
2. Copy `.env.example` to `.env` and add your Supabase URL + anon key
3. Run `supabase_schema.sql` in Supabase SQL Editor
4. `npm run dev`
